## 文件结构

```
src/data/ingredients/
├── DESIGN.md          ← 你现在看的这个
├── grades.js          ← 9 个品阶定义
├── plants.js          ← 9 种植物
├── minerals.js        ← 9 种矿物
├── liquids.js         ← 9 种液体
├── index.js           ← 汇总导出
├── 2d/                ← 2D 实现(UI 按钮 / 卡片)
│   ├── CodexPanel.js  ← 图鉴面板(2D 网格)
│   └── codex.css      ← 样式
└── 3d/                ← 3D 实现(场景里的几何)
    └── Ingredient3D.js ← 程序化生成 3D 药材
```

## 视觉差异(2D / 3D 共用)

| 档位 | 2D 表现 | 3D 表现 |
|---|---|---|
| **凡品** (1-3) | 单色背景,无辉光,无装饰 | 简单球体/八面体,无 emissive |
| **灵品** (4-6) | 渐变背景,微辉光,小光点 | 多面晶体 + 弱 emissive + 1 圈光晕 |
| **仙品** (7-9) | 动画辉光,粒子光点,强金光边框 | 复杂晶体簇 + 强 emissive + 3 圈旋转光点 + 体积光 |
